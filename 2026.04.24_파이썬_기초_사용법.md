## 1. 왜 파이썬인가

### 핵심 메시지

- 코딩을 배우는 게 아니라, **도구를 하나 장착하는 것**
- AI가 코드를 만들어주고, 실행하고 결과를 보면 됨

### 파이썬으로 할 수 있는 작업

| 작업 | 설명 |
|------|------|
| 논문 처리 | PDF에서 텍스트 자동 추출 |
| LLM 연동 | AI에게 질문하고 답변 받아오기 |
| 시각화 | 데이터를 표와 그래프로 변환 |
| 통계 분석 | 평균, 표준편차, 상관관계 등 자동 계산 |
| 엑셀 자동화 | 엑셀 파일 읽기·가공·저장 |

## 2. 환경 세팅

### Python 설치

1. `https://www.python.org/downloads/windows/` 접속
2. Ctrl + F 후 3.11 검색
3. `Python 3.11.9 - April 2, 2024`의 `Download Windows installer (64-bit)` 다운로드
2. 설치 파일 실행
3. 첫 화면에서 **"Add python.exe to PATH" 반드시 체크**
4. Install Now 클릭 후 설치 완료

### 설치 확인 (cmd 사용)

> cmd(명령 프롬프트)는 윈도우에 기본으로 설치된 텍스트 입력 창입니다.  
> `pip install` 등 파이썬 관련 명령어는 여기서 실행합니다.

1. 윈도우 키 + R
2. `cmd` 입력 후 엔터
3. 아래 명령어 입력

```
python --version
```

3. `Python 3.x.x` 가 출력되면 설치 성공

### VSCode 설치

1. `https://code.visualstudio.com/` 접속
2. Download for Windows 클릭
3. 설치 완료 후 VSCode 실행
4. 왼쪽 블록 아이콘(Extensions) 클릭 후 아래 두 가지 설치
   - `Korean Language Pack for Visual Studio Code` — 한글화
   - `Python` (Microsoft 공식) — 파이썬 실행 지원
5. VSCode 재시작

## 3. 파이썬 첫 실행

### 파일 만들고 실행하기

1. 바탕화면에 작업용 폴더 생성 (예: `python-work`)
2. VSCode 상단 메뉴 → 파일 → 폴더 열기 → 만든 폴더 선택
3. 새 파일 만들기 → 파일명은 반드시 `.py` 로 끝나야 함 (예: `test.py`)
4. 아래 코드 붙여넣기

```python
print("안녕하세요")
```

5. 터미널 열기: 상단 메뉴 → 터미널 → 새 터미널 (또는 `Ctrl + 백틱(`)`)
   - VSCode 터미널은 cmd와 같은 역할을 VSCode 안에서 하는 창입니다
6. 터미널에 아래 입력

```
python test.py
```

7. `안녕하세요` 가 출력되면 성공

### 앞으로 반복할 패턴

```
AI에게 코드 요청  →  파일에 붙여넣기  →  python 파일명.py 실행
```

## 4. pip와 라이브러리

### 개념 설명

| 비유 | 실제 |
|------|------|
| 스마트폰 | 파이썬 |
| 앱 | 라이브러리 |
| 앱스토어 | pip |

기본 파이썬만으로는 할 수 있는 것이 적습니다.  
라이브러리를 설치해야 그래프를 그리고, 엑셀을 읽고, AI를 연결할 수 있습니다.

### 설치 방법

cmd 또는 VSCode 터미널에서 아래 형식으로 입력합니다.

```
pip install 라이브러리명
```

예시:

```
pip install pandas
pip install matplotlib
pip install openai
```

### 자주 쓰는 라이브러리

| 라이브러리 | 용도 |
|-----------|------|
| `pandas` | 표(데이터) 다루기, 엑셀 읽기 |
| `matplotlib` | 그래프 그리기 |
| `scipy` / `statsmodels` | 통계 분석 |
| `pdfplumber` | PDF 텍스트 추출 |
| `openai` | ChatGPT 등 LLM 연결 |

### 오류가 날 때

1. 터미널에 출력된 오류 메시지를 마우스로 드래그해서 복사
2. AI에게 아래 형식으로 질문

```
test.py 파일에서 파이썬 오류가 발생하였는데, 해결 방법 알려줘

오류 메시지:
(복사한 내용 붙여넣기)
```

## 5. AI와 함께 쓰는 파이썬

### 코드 요청하는 법

막연하게 요청하는 것보다 구체적으로 요청할수록 정확한 코드를 받을 수 있습니다.

**나쁜 예**
```
그래프 그려줘
```

**좋은 예**
```
파이썬으로 엑셀 파일을 읽어서
A열을 x축, B열을 y축으로 하는
꺾은선 그래프를 그리는 코드 짜줘.
파일 이름은 data.xlsx야.
```

요청 시 포함하면 좋은 것:
- 파일 이름 및 형식 (엑셀, CSV, PDF 등)
- 원하는 결과물 (그래프, 표, 숫자 등)
- 어떤 열 또는 항목을 사용할지

### 오류 해결 요청하는 법

```
파이썬 실행 중 오류가 났습니다. 해결해주세요.

[코드]
(코드 전체 붙여넣기)

[오류 메시지]
(오류 내용 붙여넣기)
```

### 코드 이해가 안 될 때

```
이 파이썬 코드가 어떤 동작을 하는지
비개발자도 이해할 수 있게 설명해줘

(코드 붙여넣기)
```

## 6. Git으로 할 수 있는 것

> Git은 코드(파일)의 변경 이력을 기록하고 관리하는 도구입니다.

### 핵심 개념

- **버전 관리**: 파일을 수정할 때마다 이전 버전을 저장해두어, 언제든 되돌릴 수 있음
- **백업**: GitHub 같은 원격 저장소에 올려두면 PC가 망가져도 파일이 안전함
- **협업**: 여러 명이 같은 파일을 동시에 작업할 때 충돌 없이 합칠 수 있음

### 파이썬 작업에서 Git이 유용한 경우

| 상황 | Git 없이 | Git 있으면 |
|------|---------|-----------|
| 코드 수정 후 오류 발생 | 수동으로 백업해둔 파일 찾기 | 이전 버전으로 즉시 복원 |
| 여러 버전 관리 | 코드_최종.py, 코드_최최종.py | 버전 이력으로 깔끔하게 관리 |
| 다른 PC에서 작업 | USB 또는 이메일로 전달 | GitHub에서 바로 내려받기 |

## 7. AI 종류와 장단점

파이썬 코드 작성·오류 해결 시 활용할 수 있는 주요 AI 도구입니다.

| AI | 특징 |
|----|------|
| ChatGPT (OpenAI) | 가장 널리 알려진 AI | 
| Claude (Anthropic) | 긴 문서·코드 처리에 강함 | 
| Gemini (Google) | Google 서비스 연동 |  
| Copilot (Microsoft) | VSCode에 내장 | 

### 코딩 에이전트

일반 AI 챗봇과 달리, **코딩 에이전트**는 코드 작성뿐 아니라 파일 생성·실행·수정까지 직접 처리합니다.

| 도구 | 특징 |
|------|------|
| Cursor | VSCode 기반 에디터 |
| GitHub Copilot | VSCode 내장. 코드 입력 중 실시간 자동완성 제안 |
| Claude Code | 터미널에서 실행. 파일 읽기·쓰기·명령어 실행까지 자율 |

> 단순히 코드를 알려주는 챗봇과 달리, 코딩 에이전트는 직접 파일을 열고 수정하는 수준까지 동작합니다.
